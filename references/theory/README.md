# Theory

Theory is split by failure mode, not by historical chronology.

- `auxiliary-mass-flow/`
  AMFlow-specific foundations and direct descendants.
- `differential-equations/`
  Classical master-integral DE method, canonical bases, epsilon-form reduction.
- `boundaries-and-regions/`
  Large-`eta` boundary logic and singular-limit analysis.
- `analytic-continuation-and-singularities/`
  Physical-region continuation, complex masses, Landau/cut structure.
- `series-solvers/`
  Local series expansions, patch matching, and DE solvers closest to a rewrite.
- `precision-and-error-control/`
  Arbitrary precision and stability/error-discipline references.
